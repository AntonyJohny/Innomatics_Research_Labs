# RAG-Based Customer Support Assistant with LangGraph & HITL

## 📌 Project Overview
This project is a sophisticated Customer Support Assistant built using **Retrieval-Augmented Generation (RAG)** and **LangGraph**. It is designed to provide accurate, context-aware responses by querying a specific knowledge base (PDF) while maintaining high safety standards through **Human-in-the-Loop (HITL)** escalation.

The system is particularly robust because it doesn't just "chat"—it follows a structured graph-based workflow to decide whether it can answer a query autonomously or if it needs to pause and wait for a human supervisor.

## 🏗️ High-Level Design (HLD)
The architecture follows a modular approach:
1.  **Document Ingestion:** Uses `PyPDFLoader` and `RecursiveCharacterTextSplitter` to process raw data.
2.  **Vector Store:** `ChromaDB` stores embeddings generated by the `all-MiniLM-L6-v2` model.
3.  **Orchestration Logic:** `LangGraph` manages the state and transitions between the "Retriever" and the "Assistant" nodes.
4.  **Inference Engine:** Powered by `Llama 3.1-8B-Instant` via the Groq API for near-instant response generation.

## ⚙️ Key Features
* **Contextual Retrieval:** The assistant only answers based on the provided PDF, reducing hallucinations.
* **Safety Router:** Automatic detection of danger keywords (e.g., "fire", "smoke", "legal") to trigger immediate human intervention.
* **State Management:** Uses a `TypedDict` state to track context and escalation flags across the workflow.
* **Human-in-the-Loop:** A dedicated mechanism to pause AI output and allow a supervisor to provide a manual override for complex or high-priority issues.

## 🛠️ Tech Stack
* **Language:** Python
* **Frameworks:** LangChain, LangGraph
* **LLM:** Llama 3.1 (Groq Cloud)
* **Database:** ChromaDB (Vector Store)
* **Embeddings:** HuggingFace Sentence-Transformers
* **Environment:** Google Colab / VS Code

## 🚀 Installation & Setup
1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/AntonyJohny/Innomatics_Research_Labs.git
    cd RAG_main_project
    ```
2.  **Install Dependencies:**
    ```bash
    pip install langchain-groq langgraph chromadb pypdf sentence-transformers langchain-community
    ```
3.  **API Configuration:**
    Ensure your `GROQ_API_KEY` is set in the environment variables.
4.  **Add Knowledge Base:**
    Place your source PDF as `data.pdf` in the root directory.

## 🧪 Sample Queries
* **Standard RAG:** *"What is the warranty period for a smartphone?"*
* **Out of Scope:** *"What is the price of the new SwiftTech laptop?"* (AI will state it cannot find this info).
* **HITL Escalation:** *"My device is smoking and I'm calling my lawyer!"* (System will trigger Human Supervisor Override).
